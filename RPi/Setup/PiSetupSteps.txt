* enable SSH server:
sudo raspi-config
- Advanced -> SSH -> enable server

* setup static IP:
get mac addr for pi: ifconfig

* get apt-get up to date:
sudo apt-get update
sudo apt-get upgrade
sudo apt-get autoclean
sudo apt-get autoremove

* apt-file package finder:
sudo apt-get install apt-file
sudo apt-file update

*update kernel & firmware
sudo rpi-update

* install VNC
sudo apt-get install tightvncserver

* start VNC server:
tightvncserver
VNC full screen:	tightvncserver :1 -geometry 1366x768
VNC windows:		tightvncserver :1 -geometry 1280x720

* Bittorrent:
sudo apt-get install deluge

*Mounting network drives:
sudo apt-get install ntfs-3g
sudo apt-get install samba-common-bin
